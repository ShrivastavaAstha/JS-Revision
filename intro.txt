Why Javascript?
--> 1.Because Javascript is used and supported by all the browsers.
    2.Because Javascript can be used for both Frontend and Backend and Even Database openly supports JS.
    3.Because of huge community support of Javascript , which helps in error debugging and implementation of features.
    4.Javascript is used by top notch companies.
    5.Huge trends in Javascript in modern days.
    6.Javascript can be used with Mobile Dev(React Native), Web Dev(Mern) and many more.
    7.We need to learn Javascript because we want to learn React(Frontend) and Node(Backend).
    8.Javascript is very important for MERN Development, MERN is a subset of Full stack Development.
    M=>MangoDB (Database)
    E=>ExpressJS (Backend server)
    R=>ReactJS (Frontend)
    N=>NodeJS (Backend server)
